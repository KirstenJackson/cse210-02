Welcome! 

This is a hilo card game

You start with 300 points
The first card will display, and then you'll have to guess if the next card is higher or lower.
If you guess correctly, then you get 100 points!
If you guess incorrectly, then you lose 75 points.
You can keep playing until you run out of points, or until you get bored.

run the Program.cs file to start playing!
